# 智绘赤星 (AresVision) — 数据可视化方案

---

## 一、高视觉冲击力图表（Hero Visuals）

### 1. 3D 火星臭氧点云地球 / Mars Ozone Point-Cloud Globe

| 项目 | 说明 |
|---|---|
| **数据来源** | OpenMARS `o3col` |
| **维度映射** | 经度 → 球面 φ，纬度 → 球面 θ，臭氧柱浓度 → 点颜色（Viridis/Inferno 色阶）+ 点高度（可选凸起） |
| **科学价值** | 直观展示全球臭氧分布的空间异质性，一眼识别极地臭氧高值区与赤道低值带 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | `react-globe.gl` 自定义 Points Layer + 自转动画；或 Three.js `BufferGeometry` 粒子系统，支持 Ls 驱动逐帧刷新 |

### 2. 球面臭氧热力动画 / Spherical Heatmap Animation

| 项目 | 说明 |
|---|---|
| **数据来源** | OpenMARS `o3col`（全年 Ls 序列） |
| **维度映射** | 球面 UV 映射到 36×72 网格，颜色 → 臭氧浓度，时间轴 → Ls 0°–360° |
| **科学价值** | 动态展示火星全年臭氧的时空演化过程，可观察季节性臭氧"脉冲"从极区向中纬扩散的过程 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | Three.js 自定义 `ShaderMaterial`，将 36×72 数据写入 `DataTexture` 作为 UV 贴图；每帧更新纹理实现动画；配合 bloom 后处理增强发光感 |

### 3. 风场粒子流动图 / Wind Particle Flow Field

| 项目 | 说明 |
|---|---|
| **数据来源** | MCD `U_Wind` + `V_Wind` |
| **维度映射** | 经纬网格 → 粒子初始位置，U/V 分量 → 粒子运动方向与速度，颜色 → 风速大小 |
| **科学价值** | 展示火星大气环流格局（哈德莱环流、极地涡旋），揭示风场如何输运臭氧 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | WebGL 粒子系统（参考 earth.nullschool.net 风场可视化），使用 `requestAnimationFrame` 驱动粒子步进；或使用 `deck.gl` 的 `TripsLayer` |

### 4. 3D 地形 + 臭氧叠加图 / Mars Topography with Ozone Overlay

| 项目 | 说明 |
|---|---|
| **数据来源** | MOLA 地形数据（公开）+ OpenMARS `o3col` |
| **维度映射** | 经纬 → XY，地形高度 → Z（球面凹凸），臭氧浓度 → 颜色叠加层 |
| **科学价值** | 揭示地形（如 Olympus Mons、Hellas Basin）与臭氧分布的空间相关性 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | Three.js `SphereGeometry` + 顶点位移贴图（displacement map）+ 自定义臭氧颜色贴图混合；或 CesiumJS |

### 5. 原始-预测-差值 三联对比地球 / Tri-Panel Globe Comparison

| 项目 | 说明 |
|---|---|
| **数据来源** | OpenMARS 真值 + PredRNNv2 预测输出 + 差值计算 |
| **维度映射** | 三个并排 3D 地球，分别着色为：真值场、预测场、差值场（蓝红发散色阶） |
| **科学价值** | 核心评估图——直观展示模型在哪些区域/季节预测准确，哪些存在系统偏差 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | 三个 `react-globe.gl` 实例并排，共享相机角度（同步旋转），差值用 `RdBu` 发散色阶 |

## 二、专业科学图表（Scientific Charts）

### 6. Ls-纬度臭氧热力图 / Solar Longitude–Latitude Heatmap

| 项目 | 说明 |
|---|---|
| **数据来源** | OpenMARS `o3col`（纬向平均后） |
| **维度映射** | X → Ls（0°–360°），Y → 纬度（-90°–90°），颜色 → 臭氧柱浓度 |
| **科学价值** | 行星科学经典图式——展示臭氧季节循环的纬度依赖性，识别春季极地臭氧峰值 |
| **视觉吸引力** | ⭐⭐⭐⭐ |
| **实现建议** | Plotly.js `Heatmap` + 自定义 Viridis colorscale；叠加季节分界虚线（Ls=0/90/180/270） |

### 7. 纬度带季节趋势折线图 / Latitude-Band Seasonal Trend Lines

| 项目 | 说明 |
|---|---|
| **数据来源** | OpenMARS `o3col`（按 5 个纬度带聚合） |
| **维度映射** | X → Ls，Y → 臭氧柱浓度均值，颜色/线型 → 纬度带（极地北/中纬北/赤道/中纬南/极地南） |
| **科学价值** | 量化各纬度带臭氧季节幅度差异，验证极地 vs 赤道的反相位关系 |
| **视觉吸引力** | ⭐⭐⭐ |
| **实现建议** | Plotly.js `Scatter` 多线图，半透明置信区间带（若有多年数据） |

### 8. 昼夜臭氧变化曲线 / Diurnal Ozone Variation Curve

| 项目 | 说明 |
|---|---|
| **数据来源** | MCD 数据（sol × hour 维度）融合预测结果 |
| **维度映射** | X → 地方时（hour 0–24），Y → 臭氧浓度，分组 → 不同纬度带或季节 |
| **科学价值** | 展示光化学驱动的臭氧日变化，验证模型是否捕捉到昼夜循环信号 |
| **视觉吸引力** | ⭐⭐⭐ |
| **实现建议** | Plotly.js 折线图 + 日出日落灰色区域标注；或极坐标雷达图（24小时为一圈） |

### 9. 环境变量相关性矩阵 / Variable Correlation Heatmap

| 项目 | 说明 |
|---|---|
| **数据来源** | MCD 6 个环境变量 + OpenMARS o3col |
| **维度映射** | X/Y → 变量名（7×7），颜色 → Pearson 相关系数（-1 到 +1，RdBu 色阶） |
| **科学价值** | 揭示哪些气象因子与臭氧最相关（温度？沙尘？辐射？），指导特征选择 |
| **视觉吸引力** | ⭐⭐⭐ |
| **实现建议** | Plotly.js `Heatmap` + `annotationtext` 显示数值；或 D3.js 自定义矩阵 |

### 10. 环境变量时空面板 / Multi-Variable Dashboard Panel

| 项目 | 说明 |
|---|---|
| **数据来源** | MCD `Temperature`, `Dust_Optical_Depth`, `Solar_Flux_DN`, `U_Wind`, `V_Wind`, `Pressure` |
| **维度映射** | 6 个小热力图排列（2×3 grid），每个 X→Ls，Y→纬度，颜色→变量值 |
| **科学价值** | 同时对比多个驱动变量的季节-纬度分布，建立臭氧与环境的视觉对应 |
| **视觉吸引力** | ⭐⭐⭐⭐ |
| **实现建议** | Plotly.js `subplots` 或 CSS Grid 内嵌 6 个独立 Plotly 图；统一色阶 + 联动缩放 |

## 三、预测评估图表（Model Evaluation）

### 11. 预测误差空间分布图 / Prediction Error Spatial Map

| 项目 | 说明 |
|---|---|
| **数据来源** | 预测结果 - OpenMARS 真值 |
| **维度映射** | 等距圆柱投影，X→经度，Y→纬度，颜色→RMSE 或 MAE（发散色阶） |
| **科学价值** | 定位模型误差的空间热点（极地？地形区？）引导模型改进方向 |
| **视觉吸引力** | ⭐⭐⭐⭐ |
| **实现建议** | Plotly.js `Heatmap` 或 Three.js 2D 平面投影，叠加火星地形轮廓线 |

### 12. 逐步预测衰减曲线 / Step-wise Prediction Decay

| 项目 | 说明 |
|---|---|
| **数据来源** | 预测结果（horizon 1/2/3）vs 真值 |
| **维度映射** | X → 预测步数（1,2,3），Y → RMSE/SSIM/MAE，分组 → 不同变量组合 |
| **科学价值** | 量化预测精度随步数的衰减速率，评估模型的可靠预测窗口 |
| **视觉吸引力** | ⭐⭐⭐ |
| **实现建议** | Plotly.js 分组柱状图 + 误差棒；或折线图带置信区间 |

### 13. 变量消融实验对比图 / Ablation Study Comparison

| 项目 | 说明 |
|---|---|
| **数据来源** | 不同变量组合的预测结果 |
| **维度映射** | X → 变量组合（如 "全部"/"去沙尘"/"仅温度+辐射"），Y → 评估指标 |
| **科学价值** | 直接回答"哪些变量对预测最关键"——沙尘？温度？辐射？ |
| **视觉吸引力** | ⭐⭐⭐ |
| **实现建议** | 分组柱状图或平行坐标图（Plotly `parcoords`） |

### 14. 预测动画回放 / Prediction Playback Animation

| 项目 | 说明 |
|---|---|
| **数据来源** | 预测结果序列 + 真值序列 |
| **维度映射** | 双面板同步动画：左侧真值、右侧预测，时间轴驱动 Ls 递进 |
| **科学价值** | 动态展示模型"跟踪"真实臭氧演化的能力，直观评估时序一致性 |
| **视觉吸引力** | ⭐⭐⭐⭐⭐ |
| **实现建议** | 两个同步 Globe + 共享 `TimelinePlayer`，底部 Ls 进度条同步驱动 |

## 四、三联视图方案说明

针对核心评估场景，推荐以下三联（Triptych）布局：

```
┌─────────────────┬─────────────────┬─────────────────┐
│   原始真值场     │   模型预测场     │   差值场         │
│  (OpenMARS)     │  (PredRNNv2)    │  (预测-真值)     │
│                 │                 │                 │
│  Viridis 色阶   │  Viridis 色阶   │  RdBu 发散色阶   │
│  [0, max]       │  [0, max]       │  [-δ, +δ]       │
└─────────────────┴─────────────────┴─────────────────┘
               ◄── 共享时间轴 (Ls) ──►
```

**实现形态**：
- **3D 版**：三个 Globe 并排，同步旋转（图表 #5）
- **2D 版**：三个 Ls-纬度热力图并排，同步缩放（基于图表 #6 扩展）
- **混合版**：上方一个大 Globe 可切换 Tab（真值/预测/差值），下方三个小热力图并排

**交互设计**：
- 鼠标悬停任一面板的网格点，三个面板同时高亮对应位置并显示数值 tooltip
- 时间轴拖动时三面板同步更新
- 支持"锁定差值阈值"滑块，高亮差值超过阈值的区域
